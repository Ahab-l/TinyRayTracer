# 学习经历
主要跟随三部曲学习，并完成了一个简易的光追渲染器，光追这种基于采样的渲染方式可以很轻松的渲染出高质量的图像，但是往往需要大量的时间采样降噪。因此该项目的重点在于如何加速光线追踪，从BVH到蒙特卡洛方法都为此服务。
